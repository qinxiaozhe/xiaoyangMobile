招聘职位 : 开发工程师

工作职责
1、 充分理解产品和项目需求，据此进行开发设计；
2、 参与公司产品、项目的代码开发；
3、 解决整体系统在开发、运行中出现的各种问题
职位要求
1、开发语言不限，但优先精通.Net技术；
2、3年以上程序开发经验；
3、 熟练掌握JavaScript、HTML/HTML5/XML、CSS/CSS3、Ajax、JSON等前端开发技术，熟悉 SQL Server 数据库
4、具有良好的代码复用性、模块化思维能力；
5、具有良好的规范化、标准化的代码编写习惯以及文档整理习惯；
6、对工作严格要求，认真负责。具有很强的自学能力、分析能力、解决问题的能力；
7、具有良好的团队精神和协作能力；

招聘职位 : 测试工程师

主要职责
1、在公司的安排和指导下，完成产品和项目的测试计划工作；
2、掌握较深层次的测试方法、测试技术和较复杂的业务流程，按时、按质地进行测试工作；
3、 协助编写版本发布文档、《用户手册》、《操作手册》；
4、增强个人团队合作意识，协助测试经理及开发部门完成测试工作；
5、了解项目的基本流程，配合项目其他人员保证项目的顺利完成。
职位要求
1、2年以上测试经验，本科以上学历，英语CET- 6级或同等以上水平；
2、熟悉网站网页测试，熟悉常用数据库操作；
3、 熟悉Windows和Mac系统使用，熟悉各种浏览器的配置和区别；
4、了解软件工程和软件过程，了解Scrum开发模式；
5、了解HTML、JavaScript、CSS、SQL、HTTP、TCP/IP、SSL、LDAP；
6、了解VPN、Proxy Server、Web Server、Firewall；
7、了解常用编程语言，如C# 或VB，能够编写自动化测试脚本。
